Learning Python

This reserach project took place during the summer of 2022. A group of high school students were brought to Lawrence Berkeley National Lab to learn data science with Python. I helped teach this along with developing a Jupyter Notebook where students analyze data on galaxies and supernovae.

This project culminated in a research poster that was presented live. All analysis was done in GoogleSheets.
